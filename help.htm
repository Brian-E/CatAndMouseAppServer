Help/Instructions for Cat and Mouse

The worlds largest game of…well…cat and mouse!

 

Thank you downloading and installing this application.  This tutorial will provide you with instructions on how this game works, how to play, navigating the UI and how to create your own games!

How to Play

Cat and Mouse is a game of chase/catch where players who are ‘cats’ are trying to catch the players in the game that are ‘mice’.  It uses the Bluetooth technology in your device/phone to recognize when a cat has ‘caught’ a mouse; which occurs when your devices/phones are within Bluetooth range of each other (typically 20 feet).  Device pairing is not required!  Points are earned for cat players when they catch mouse players, and mouse players earn points for each minute they’re in the game and do not get caught.  Your score is cumulative for the entire length of the game, and the winner is whoever has the most points at the end of the game. 

Becoming a Cat/Mouse

The game will randomly assign players in the game to become a cat, and provide you a notification when this occurs.  The notification will take you to the Game Manager screen where you can click the ‘Get Mice’ button, which will prompt you to place your device in Bluetooth Discoverable mode for 5 minutes.  This is required for cat players to catch mouse players.  Each cat player gets a 5 minute turn to try and catch as many mouse players as they can.  You can only catch the same mouse player once per turn.  When your 5 minutes are up, you’ll receive another notification indicating that you have changed back to a mouse, and the game will select another random player to be a cat.

Ratios

When a game is created, a ratio value is set which determines how many cats there will be in the game.  For example, if a ratio of 5:1 is set, then there will be 1 cat player for every 5 mouse players.  The game server will assign more cats (or not) as needed based on this ratio and the current occupancy of the game.  This ratio is also critical to scoring, which is discussed next.

Scoring

As mentioned earlier, points are scored in the game when cat players catch mouse players, as well as mouse players accumulating points for each minute they are in the game and stay ‘alive’.  Mouse players get 1 point for each minute they are in the game and do not get caught. If you get caught, you will not accumulate mouse points again for 5 minutes.  For cat players who catch a mouse player, they are awarded a point value that is the product of the game ratio times 5.  For example, if you are in a game with a ratio of 4:1 and you catch a mouse, then you will be award 4x5=20 points for that mouse.  The logic behind this formula is that each cat turn lasts 5 minutes, and the greater the ratio, the longer you will have to wait to become a cat, and so naturally it is then worth more to catch a mouse. 

Notifications

Cat and Mouse will annoy you with several notifications to alert you when you become a cat, or get caught as a mouse.  This way you don’t have to stare at the screen the whole time to play the game; just let the game alert you when something interesting happens.  These notifications are configurable within the Settings menu.

Game Modes

Now that you know how the game works, it’s time to mess with your head and introduce a different game mode.  Cat and Mouse supports 2 different game modes: Normal and Reverse.  Everything you’ve read up until now applies to the Normal game mode.  Reverse mode is slightly different, but many of the same principles apply.

Normal Mode

In a Normal game, players join the game as a mouse, and most of the players in the game will be a mouse, depending on the game ratio.  As mentioned, you’ll receive a notification when you become a cat, as well as when you catch a mouse, and mouse players will get a notification when they get caught.  How often you become a cat is dependent on the ratio, as well as if there are enough players in the game.  For example, if you’re in a game with a 5:1 ratio, then you should be chosen as a cat roughly every 25 minutes (if there are at least 6 players in the game).  This game mode is designed to be a bit more relaxed; there isn’t much to do when you’re a mouse (except avoid cats!), so you can stick your phone back in your pocket and go about your business.  When you get your cat notification, then it’s time to catch mice!  Get to the Game Manager screen, click on the ‘Get Mice’ button, accept the Discoverable mode prompt, then try and find some cats.  Use the Player Map to locate mouse players near you and try to get close enough to catch them!

Reverse Mode

In a Reverse game, everything is…reversed!  Players join the game as a cat, and the game server will randomly assign players to be a mouse based on the ratio, and will spend 5 minutes as a mouse.  Catching a mouse works the same way – just get within Bluetooth distance of the mouse player (Cat players need to stay in discoverable mode).  Scoring works like this: mouse players receive points for each minute they stay alive, and that value is the product of the game ratio times 5.  For example, in a game with a ratio of 4:1, a mouse player will receive 4x5=20 points for each minute they do not get caught.  If a mouse player gets caught, then their turn will be over, and the game will assign a new mouse.  Cat players score points only by catching mice, and they receive 5 points for catching a mouse.  This game mode is faster paced and more intense, as you will spend most of your time as a cat trying to catch mice, and then taking turns being a mouse and trying to hide and stay alive!

 

Player Map

The Game Manager screen contains a player map which will show you the locations of all the players in the current game, as well as whether they are a cat or a mouse. These icons are also clickable to retrieve Feint information on the user, such as their name, picture, and score. 

Feint Integration

Cat and Mouse is integrated with features from OpenFeint to handle user names, leaderboards, and achievements that can be recognized for accomplishments within the game.
